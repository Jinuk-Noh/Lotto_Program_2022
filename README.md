## 보완할 점
1. 회차 클릭 시 텍스트가 자동으로 비워지고 번호를 바로 입력할 수 있도록 수정 필요
2. 번호 입력에 숫자만 입력되도록 제한
3. 입력한 숫자 중복 확인 및 유효성(1~45 사이 숫자만 입력) 체크 필요
